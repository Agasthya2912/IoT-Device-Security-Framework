# IoT Device Security Framework

A multi-layered IoT security framework implementing:

- JWT-based authentication
- AES-256 encryption
- SHA-256 hashing
- DoS attack prevention using rate limiting

## Features
- Secure device authentication
- Encrypted data transmission
- Brute-force & DoS mitigation
- Web-based interface using Flask

## Technologies
Python, Flask, JWT, AES-256, SHA-256
